# ConNet
## Introduction
The code of "Iterative Joint Learning Integrating Temporal and Geometric Information for Vessel Segmentation in X-ray Coronary Angiography" in Medical Physics 2025.
- The code is based on: Python 3.8 + pytorch 1.11.0.
- Config-driven experiment management
- Support for custom datasets and models
